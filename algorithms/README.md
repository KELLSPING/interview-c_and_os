## 介紹 ##

* Problem-solving procedure.
  * 解決問題的過程
* 一個"有限的"、"明確的"、"有效的過程"、並且"有輸出"
* 問題
  1. input
  2. output
* 解決方法
  1. 明確性 (definiteness) : 必須是明確的、不含糊的。
  2. 有限性 (finiteness) : 終止於有限步數。
  3. 有效性 (effectiveness) : 用紙和筆畫出，必須是基本的且可行的。
  4. 過程 (procedure) : 按邏輯順序排列的特定步驟。
